  • How do I find out more?
    • See the website:www.cs.toronto.edu/~arnold/cit
      Email Arnold Rosenbloom at arnold@cs.toronto.edu .
  • What questions does CIT advanced answer?
    • I am tired of programs with a single large main method, what should I do? What is the Object Oriented Paradigm and how do I use it? How do I draw, manipulate images and write games in Java? What is recursion and how do I effectively use it? How do I teach my students all of the above?
  • Who is CIT advanced for?
    • Teachers with some exposure to Java. You should understand if, while, for loops and arrays and can write, compile and execute simple Java programs. No object oriented knowledge will be assumed. A beginner section (for teachers with no exposure to Java) is also available, see http://www.cs.toronto.edu/cit
  • What happens at CIT advanced?
    • A typical day will have you alternating between interactive lectures and solving posed programming problems in our lab.
  • What do I take away with me?
    • A collection of exercises fully understood and solved by you and your colleagues for use in your classroom. All lecture notes and sample code for the course. Practical experience with Java and the OO Paradigm. An approach to teaching Java and the OO Paradigm.
Program Topics
  • Basic Tools:
    • Eclipse, the JDK, the Java API
  • Non-Object Oriented Java:
    • A quick review of HelloWorld, primitive types (int, double, etc.), expression, operation and operator, converting and casting, if, while, for, screen and file I/O, methods & recursion, arrays and collection classes
  • Object Oriented Java:
    • The object-oriented paradigm, Java libraries and APIs, References, Arrays, Strings, Visibility: public, private, protected, Inheritance and overriding, toString, equals
  • Using the OO Paradigm:
    • Identifying and using classes, Instance vs static, Constructors, Overloading & overriding, Abstract classes & interfaces, Design Patterns
  • Other topics:
    • Drawing, painting, image manipulation, sound and animation, GUI Programming (buttons, text fields, check boxes), Game programming, Network programming & Multithreadding, Software engineering tools
